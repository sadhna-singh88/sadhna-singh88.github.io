Display Inline vs Inline Blocks!

What is Display Property ?

The display property is one of the most important features in CSS.This property specifies the display behavior of an element, or we can say it controls the layout of an element (so basically how an element will look its dependents on display property ) There can be other values related to this property in CSS. “Inline” and “Inline-Block'' are two such values.

Display: Inline

Just like its name “inline”, this feature will display en elements in the same line. It does not start with a new line.They don't accept width or height properties and top-bottom margin. they take only as much space as they need so if we set any width and height value,they will not have any effect. Here are a few elements that have a default inline property:

  • <a>
  • <label>
  • <span>

Display: Inline-Block

As we can understand from its name, display: inline-block will display both the characteristics of inline and block-level elements. Inline-block is essentially the same thing as inline, except that we can set height and width values.display:inline-block does not add a line-break after the element. So, the element can sit next to other elements. Here are a few elements that have a default inline property:

  • <button>
  • <img>
  • <input>
